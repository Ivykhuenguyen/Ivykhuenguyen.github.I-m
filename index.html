Why is this skill important?

Doing the dishes by hand is exhausting and time consuming, but that problem is solved by the dishwasher! With the dishwasher, you can save a lot of time, and it is proven to be more efficient according to users as well! That requires some skill and knowledge though, because the dishwasher can be quite complicated, lucky for you, this website is going to teach you how!

REQUIRED ITEMS

  • A dishwasher
  • Soap/detergent
  • Dishes to wash

STEPS

STEP 1

Open the dishwasher,pull out the racks

STEP 2

Put the bowls and the cups on top rack. Make sure there is enough space between the items so the water can go through. Place the items upside down.

STEP 3

Close the upper rack, now start putting the plates, the pots and their lids, the pans on the bottom rack.

The dishwasher cannot wash all the food pieces off the plates/bowls, so you should rinse off all the food pieces before you put it in the dishwasher!

STEP 4

But what about the cutlery? There is a small basket that is placed inside the bottom basket (where you put the plates previously), and you can put the cutlery in there.

STEP 5

when you have finished putting all the things in, close the dishwaher, and put the soap pod/dishwasher detergent (there is a door for each type). Now adjust the washing preference (how much time, how much soap, temperature). Click start

CONGRATS, YOU HAVE DONE IT

If your dishes are clean after you use the dishwasher, and if you have an easier time doing the dishes by using it, you have excelled this skill! However, another part of using the dishwasher is how to logically organize all the things so that it all fit in the dishwasher. This website cannot teach you that, but overtime, you gain more experience and can use the dishwasher like a professional!
